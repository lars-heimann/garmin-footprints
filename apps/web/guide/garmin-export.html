Garmin account export

Get the ZIP that Run Maps can process.

Run Maps needs the full Garmin account export, not a single activity file. Garmin prepares this as a ZIP archive after you request your account data.

Steps

  1. Log into Garmin Connect.
  2. Open Garmin's export page: garmin.com/account/datamanagement/exportdata.
  3. Request your account export.
  4. Wait for Garmin to prepare it. Garmin may take up to 30 days. In practice, many exports arrive much sooner, sometimes within about 20 minutes.
  5. Download the ZIP Garmin provides.
  6. Upload that ZIP unchanged on the upload page.

Upload The Right File

  • Upload the full Garmin account export ZIP.
  • Do not unzip it.
  • The current direct-upload limit is 1 GB.
  • Do not upload a single .fit, .gpx, .tcx, or activity export.
  • Do not upload a Strava export.
  • If Garmin gives multiple files, upload the main account export archive containing DI_CONNECT.

Privacy

The ZIP contains private health and location data. Run Maps extracts only running GPS tracks, trims starts and finishes, deletes the raw ZIP after processing succeeds or fails, and publishes the generated map assets at your chosen slug URL.

Troubleshooting

  • If you see a single activity file error, request the full account export instead of exporting one activity.
  • If the ZIP does not contain Garmin account export folders, check that the archive includes DI_CONNECT.
  • If no running GPS activities are found, confirm your Garmin account contains outdoor runs with GPS tracks after the configured start date.
  • If the ZIP is larger than 1 GB, this version cannot upload it directly yet.
  • If Garmin gave you a placeholder file from iCloud or Dropbox, download the real ZIP locally first.
  • If one upload is already processing for a slug, wait for it to finish before uploading another ZIP.

FAQ

Can I upload a Strava export?
No. This version only accepts Garmin account export ZIP files.
Are generated maps private?
No. Anyone with the slug URL can view the generated map.
Is the raw Garmin ZIP kept?
No. The Worker deletes it after processing succeeds or fails, and a reaper retries deletion if needed.
