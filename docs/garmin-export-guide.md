# Garmin Export Guide

1. Open Garmin Connect in a browser.
2. Go to account data export and request your account archive.
3. Wait for Garmin to prepare the archive.
4. Download the account export ZIP.
5. Upload the ZIP unchanged in the Garmin Footprints app.

The app processes running activities with GPS tracks, trims the start and finish of each run, generates static map assets, and deletes the raw ZIP after processing succeeds or fails.
